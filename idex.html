HTML - Linguagem de marcação de hipertexto

Estructura Básica

Documento HTML São divididos em algumas partes básicas.
Antes de iniciar os elementos, devemos utilizar uma declaração <!DOCTYPE html> na primera linha.

O elemento raiz "Html" é de onde partem todos os outros elementos. Partes"Head" e "Body".

Tags e elementos

Na marcação HTML, cada elemento é representado por uma tag de abertura, o conteudo e uma tag de fechamento. Tag de abertura são escritas no formato <Nome-da-tag- >e tags de fechamento </Nome-da-tag->

Elementos de texto

Alguns elementos de texto que podemos utilizar:

  • Paragrafo - tag <p>
  • Titulos - tag <h1> a < h6>
  • Links - tag <a>

Criando uma nova página

Para criar uma nova página HTML devemos:

  1. Criar um novo arquivo com a extensão .html
  2. Inserir a declaração doctype na primeira linha
  3. Criar os elementos Html. head e body
  4. Inserir o conteúdo
